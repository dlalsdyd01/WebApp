
# 간단한 웹 애플리케이션 코드만들기  
이름과 번호, 생년월일을 받아 addbook.txt파일에 csv형식으로 저장하는 어플리케이션



터미널에서 가상환경 만들기  
```conda create -n webApp python=3.9```  


Flask 프레임워크 설치  
pip install flask
